【简介】

本软件为DMM游戏《美少女花骑士》适配的应用程序，用来取代浏览器进行游戏。
同时对部分游戏数据做出显示，并提供日志记录、提醒设置等辅助功能。


【系统需求】

Windows 7/8/8.1/10系列操作系统，32位/64位均兼容。
其中Windows 7/8/8.1系统需要安装.NET Framework 4.5.2以上版本后才能使用，Windows 10系统可以直接运行。
推荐将系统的IE浏览器更新到10以上版本，IE浏览器版本较老可能会导致一些功能不可用或者显示不正常。
本软件依赖IE，所以部分设置也继承IE。如果遇到Flash无法载入问题，请先安装IE的Flash插件，以及调整IE的相关设置，如取消屏蔽等。

.NET Framework 4.5.2官方下载地址：https://www.microsoft.com/zh-CN/download/details.aspx?id=42642


【使用说明】

1.第一次运行的时候，请先进行设置，在设置好点击“保存”后，再点界面上的“重新载入”按钮加载游戏。
2.“自动推图”功能仅在进入副本地图后才能使用。开启后会自动模拟鼠标点击前进按钮和释放阳光大招，在到达终点结算时自动终止。您也可以在任意时间再次点击该按钮停止自动操作。
3.如有任何疑问或者反馈软件问题，请至百度贴吧“美少女花骑士吧”发帖。


【开发环境】

本软件使用C#语言开发。
开发和测试环境为Windows 10 Pro 64位 + Visual Studio 2015。

【常见问题】

Q：应该如何设置代理？
A：岛风GO和ACG选择第三项，服务器填写127.0.0.1，端口岛风GO填写8099，ACG填写8123（默认端口情况下，如果您调整了端口号请变更为对应的值即可）；如果使用VPN或者改HOSTS直连，请选择第一项。第二项一般用不到，如果您选择第一项和第三项都无法连接的情况下可以尝试使用。

Q：为何能正常显示游戏界面，但是软件读取不到游戏数据？
A：这种情况一般是您当前系统中的.NET Framework版本过低导致的，请安装.NET Framework 4.5.2以上版本即可解决。另外，代理配置错误也可能会导致读取不到游戏数据，请检查设置。

Q：可以登录但是显示不出游戏界面？
A：可能是您的IE设置屏蔽了FLASH导致的。本软件基于IE，各种IE的设置不对会导致工作不正常，请调整IE设置或者重置IE设置后试试。另外Windows 7用户需要安装IE的Flash插件才能正常显示游戏界面。
